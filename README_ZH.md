# 中国农业大学研究生课表
中国农业大学研究生的一款课表显示整理程序 （目前只有网页版本，app开发中）

#### [English Version](./README.md)

## 网页端
- [测试站点](https://schedule.cabbsir.com)
- Flask (后期将会改成golang)
- Bootstrap
- Mysql

## Android APP (开发中)

## 数据获取
- 获取cookie
- 拿着cookie获取验证码
- 教务用户名 密码 验证码 模拟登录
- 登陆成功 跳转课程表页面解析数据入库
- 上一步中没有标出时间的课程，去开课时间中查询并入库
- 显示即可
